Personal Projects

Cost Estimation for Structural Engineers

I worked on this application for a civil engineering firm in Ethiopia. An application with an intuitive UI for cost estimation of structures - foundations, columns, beams and slabs. It is a complete suite that factors all the components of structures for cost estimation purposes. Features include multiple counts of components, and edit and delete features.GitHub Repo.(Java, JavaFX, SceneBuilder)

Leadership Academy Website

I worked on a team of three to develop an MVC C# driven website for our scholarship academy at school. This website suppports event registration, scholarship candidate evaluation and user management. I worked mainly on design and the user interface. Visit the website.(MVC, C# .NET, HTML, CSS, Photoshop)

All Docs

This is currently in progress. It is a document/file viewer application for Windows 8. It allows you to view your combined files from cloud storage services. Services supported now are Box, Dropbox, SkyDrive and Google Drive. I am hoping to add support for Android, Windows Phone and iPhone. GitHub Repo.(C#, XAML)

Koba

Koba is a fashion brand I'm working on with Yeamlaksira Worku, its founder. It is currently in the branding phase. The brand is inspired by Ethiopia, its rich culture, nature and beautiful people. We're looking forward to launching this brand soon.(Photoshop, Illustrator)

EOD Robot

This was at the eXtreme Engineering Competition at the SHPE 2012 Conference. In a team of ten, and within 24 hours, we designed and built a functional prototype of a robot that would be used in safe bomb extraction in the military. I proposed the robot design and made marketing materials used at the presentation. I also built a functional website for the project for presentation purposes.(Photoshop, HTML, CSS, Hardware)

Luau

This application was built with two other team members at the Fall 2012 hackathon at IIT. It is an event finder augmented reality application using the GPS and accelerometer of a smartphone. It was built as an android application using the Google Places API. It will be cleaned up in the future for other platforms. GitHub Repo.(Java)

4BYTE

I started this company in March 2008, while in high school in Ghana. It was essentially a media company that offered services in design and branding and web development. Products included magazines, t-shirts, letterheads, postage stamps, courier services and an electronic library management system. We serviced over 20 schools around the country and had a reputation of being awesome. The company is still active and serves as a design incubator for high school students in Ghana.(Photoshop, Illustrator, CSS, HTML, PHP)
